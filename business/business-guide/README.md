# business-guide

## 用途
路由商業管理相關請求到正確的專業 skill。涵蓋 C-suite 顧問、行銷、產品管理、財務、法規合規、專案管理和業務成長。

## 觸發方式
當用戶詢問商業策略、管理、行銷、財務、合規、專案管理或業務成長相關問題時觸發。

## 使用步驟
1. 識別請求類型
2. 查找 Routing Table 找到對應的 skill
3. 說明將要路由到的 skill 功能
4. 執行路由

## 規則
### 必須
- 路由到最具體的可用 skill
- 路由前說明目標 skill 的功能
- 跨領域請求時建議 skill chain

### 禁止
- 直接執行工作，必須路由
- 未經用戶確認同時啟動多個 skill

## 範例
### 正確用法
用戶：「幫我準備董事會報告」
→ 路由到 `c-suite/board-deck-builder`

### 錯誤用法
用戶：「幫我做行銷」
→ 直接開始寫文案，沒有路由到正確的行銷 skill
