# board-meeting

## 用途

幫助高階主管準備、主持及追蹤董事會會議——涵蓋議程設計、董事會動態、困難對話、治理最佳實踐及會後行動管理。

## 觸發方式

當使用者尋求董事會會議準備、設計董事會議程、管理董事會動態、處理困難的董事會對話、建立董事會治理或管理會後跟進等相關幫助時觸發。

範例觸發關鍵詞：董事會會議、董事會準備、董事會議程、董事會治理、董事會動態、董事會關係、董事會溝通、主持董事會會議。

## 使用步驟

1. **釐清會議背景** — 詢問董事會組成、會議類型及所需支援的具體面向（最多一輪）。
2. **找出主要挑戰** — 這是關於會前準備、會中動態、會後跟進，還是持續的董事會關係管理？
3. **套用董事會治理框架** — 運用：受託責任原則、董事會會議設計模式、決策權框架、資訊權協議、董事會關係培養實踐。
4. **提供角色特定指導** — CEO、CFO 和幕僚長在董事會會議中各有不同職責，建議需匹配角色。
5. **處理人際動態** — 董事會會議不只是治理練習，涉及關係、權力、信任和自我。需說明人際維度。
6. **起草具體產物** — 在相關時：議程範本、預讀結構、跟進郵件草稿、一對一準備指南。

## 規則

### 必須
- 至少在會議前 48 小時發送預讀材料
- 絕不在會議上就敏感議題讓董事會成員措手不及
- 在進入下一個議程事項前明確重申每個決策
- 跟進會議中指定的每個行動事項
- 在正式會議之外培養董事會關係

### 禁止
- 將可以作為預讀的更新用於會議時間
- 讓會議在沒有做出決策的情況下結束
- 讓困難的董事會動態不加處理地持續
- 將治理視為清單項目而非信任建立練習
- 在重大事項上對不同董事會成員進行差異化溝通

## 範例

### 正確用法

**使用者：**「我三天後有董事會會議。我的一位董事會成員在最近兩次會議上對我的領導力提出批評，我擔心這次會演變成對抗。我該怎麼辦？」

**回應：**
*會議狀況：* 一位董事會成員的批評模式，加上距離會議只有 3 天，需要在會議前採取行動，而非在會議中被動應對。

*建議：* 今天就打電話給他們，不是明天。

會前一對一有三個目的：（1）了解批評背後的真正驅動因素；（2）給他們一個不需要在眾人面前表演的表達渠道；（3）通過主動接觸而非迴避展示領導力成熟度。

*通話指南：*
「我想在董事會會議前聯繫一下。我在我們最近幾次會議中感受到您的一些擔憂，我想確保我充分理解您的觀點——不只是在會議室中。您最想討論[具體議題]的哪些方面？」

*下一步行動：*
1. 今天打電話給這位董事會成員。
2. 傾聽具體關切（績效、策略、人員）或一般性信心喪失。
3. 若擔憂合理，在董事會會議中在他們提出之前就主動說明。

### 錯誤用法

**使用者：**「如何主持好的董事會會議？」

**回應：** 好的董事會會議應該有清晰的議程、充分的準備，包括財務更新、策略討論和問答時間⋯⋯

*（通用建議、無具體情境、無董事會動態、範本式建議）*
