# copywriting

## 用途

為落地頁、銷售頁、廣告、產品描述及行動呼籲撰寫說服性、轉換導向的文案，透過解決欲望、異議與信任要素驅動讀者採取行動。

## 觸發方式

當使用者需要轉換導向的文案——落地頁、銷售電子郵件、廣告文案、產品頁面、標題、CTA，或任何以說服為主要目標的文字時觸發。

## 使用步驟

1. 釐清這份文案必須完成的唯一任務（單一轉換目標）。
2. 識別讀者的主要痛點或欲望。
3. 根據情境選擇文案框架：
   - AIDA（注意、興趣、欲望、行動）——適用落地頁
   - PAS（問題、激化、解決）——適用電子郵件和廣告
   - FAB（功能、優勢、效益）——適用產品描述
   - 前後橋接——適用轉變型文案
4. 撰寫標題：承諾具體效益或呼籲特定受眾。
5. 撰寫正文：以最有說服力的效益開頭，以證明佐證，處理主要異議。
6. 撰寫 CTA：行動動詞 + 具體效益 + 真實緊迫性（如「立即開始免費試用——無需信用卡」）。
7. 審查：刪除任何不值得存在的詞語，精簡每個句子。

## 規則

### 必須

- 以效益而非功能為先——對每個提及的功能都回答「那又怎樣？」
- 使用具體數字和具體語言（不是「節省時間」，而是「每週節省 3 小時」）
- 在文案中至少處理一個主要異議
- 文案的措辭須符合目標受眾描述自身問題的方式

### 禁止

- 在無事實依據下使用最高級（「最好的」、「最強大的」）
- 撰寫只說「點擊這裡」或「提交」的 CTA
- 將價值主張埋在落地頁折疊線以下
- 製造虛假的緊迫性或稀缺性宣稱

## 範例

### 正確用法

標題：「停止因跟進緩慢而流失潛在客戶——10 分鐘內自動化你的銷售外展。」
CTA：「開始 14 天免費試用——無設置費，隨時取消。」

### 錯誤用法

標題：「歡迎蒞臨我們的網站！」
CTA：「提交」
