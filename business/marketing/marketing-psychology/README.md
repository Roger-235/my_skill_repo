# marketing-psychology

## 用途

將行為心理學原則——社會認同、稀缺性、錨定效應、損失厭惡、互惠原則、認知偏誤——應用於行銷文案、UX 流程與轉換策略，以提升說服力、建立信任並促進行動。

## 觸發方式

當使用者詢問「將心理學應用於行銷」、「使用行為原則」、「讓文案更有說服力」、「解釋為何用戶不轉換」，或請求心理學導向的行銷改善時觸發。

## 使用步驟

1. 診斷心理障礙（用戶被要求做什麼？是什麼信念、恐懼或摩擦阻礙了行動？）
2. 應用相關原則（社會認同、稀缺性與緊迫感、錨定效應、損失厭惡、互惠原則、承諾與一致性、認知流暢度）
3. 重寫或標注素材（提供改善版本，並以內嵌標注解釋每個應用原則）
4. 識別風險（標記任何應避免的暗黑模式或操控性應用）

## 規則

### 必須

- 為每個改變引用具體原則
- 區分說服（道德）與操控（不道德）
- 建議進行 A/B 測試以驗證任何心理學干預
- 社會認同元素必須使用真實數據或具體資訊

### 禁止

- 應用虛假稀缺性或捏造的社會認同
- 使用掩蓋取消、定價或同意選項的暗黑模式
- 推薦利用脆弱用戶的認知操控手法
- 不解釋推理就應用原則

## 範例

### 正確用法

使用者：「我們的定價頁轉換率為 2%。以下是文案：[貼上內容]。」

回應：診斷錨定問題（最便宜方案排第一），改以高階方案為首，加入具體客戶數量社會認同，將 CTA 從「開始免費試用」改寫為「加入 8,400 個團隊——免費 14 天」，並標注每個改變所應用的原則。

### 錯誤用法

使用者：「讓我們的結帳流程轉換率更高。」

回應：「加個倒計時器寫著『僅剩 3 個名額！』」—— 無診斷、虛假稀缺性、無道德考量。
