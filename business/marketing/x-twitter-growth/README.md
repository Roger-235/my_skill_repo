# x-twitter-growth

## 用途

建立系統化的 X/Twitter 增長引擎，透過最佳化內容、策略性互動與受眾開發，打造具影響力的個人或品牌帳號，並將粉絲轉化為業務價值。

## 觸發方式

當使用者想要增長 X/Twitter 追蹤數、提升推文互動率、在 X 上建立個人品牌，或制定 Twitter 內容策略時觸發。

## 使用步驟

1. 最佳化個人資料：大頭貼、封面圖、個人簡介（關鍵字豐富、價值主張清晰、附 CTA 或連結）、置頂推文。
2. 定義內容支柱：持續深耕的 3–5 個主題領域。
3. 設計推文格式輪換：意見觀點、操作指南串文、清單、反向觀點、個人故事、數據分享、問題推文。
4. 建立串文框架：勾子推文（大膽宣告或承諾）→ 5–10 個編號洞察 → 摘要推文 → CTA 推文。
5. 制定互動策略：在發布自己的內容前，每天先回覆利基領域中 10–20 個帳號。
6. 定義發布節奏：每天 1–3 則推文 + 每週 1–2 個串文。
7. 識別要互動的目標帳號：領域內的頂級創作者（他們的回覆區也是你的受眾）。
8. 策略性使用引用推文：加入真正的觀點，而非只是「很棒的文章」。
9. 每週追蹤：曝光次數、個人資料訪問、粉絲增長、熱門推文。

## 規則

### 必須

- 每個串文的勾子推文必須足夠有力，能作為獨立推文使用
- 在廣播前先互動——回覆優先策略可提高觸及率
- 保持一致的發布——停止發布 3 天損害演算法分發
- 觀點需具體且有據可查，而非泛泛而談

### 禁止

- 用關注後取消關注的方式獲取粉絲
- 在沒有實質內容的情況下發布互動誘餌（「如果同意請轉推」）
- 竊取或輕微改寫他人的爆紅推文
- 自動私訊新粉絲推廣訊息

## 範例

### 正確用法

串文勾子：「我分析了行銷領域的 100 個爆紅串文。以下是它們共有的 7 個結構模式：」——具體、有數字、承諾具體價值。

### 錯誤用法

串文勾子：「我對行銷有一些想法，我認為你會覺得有趣。以下是我多年來學到的東西：」
