# landing-page-generator

## 用途

生成高轉換率的產品登陸頁文案，包含標題、副標題、價值主張、功能區塊、社交認同與行動呼籲（CTA）。

## 觸發方式

當使用者需要以下協助時觸發：
- 撰寫或改善產品行銷登陸頁
- 起草英雄區塊（Hero Section）文案
- 設計功能描述、定價頁或 CTA 文案
- 為產品行銷活動建立落地頁

## 使用步驟

1. 確認轉換目標（主要 CTA）與進入頁面的受眾
2. 撰寫英雄區塊（聚焦問題或結果的標題、副標題、CTA 按鈕文字）
3. 撰寫價值主張區塊（三個效益支柱）
4. 開發功能區塊（每個功能以效益為導向的標題 + 說明）
5. 建立社交認同區塊（客戶見證、標誌、關鍵數據）
6. 撰寫「如何運作」區塊（三步驟從註冊到獲得價值）
7. 解決常見疑慮（3–5 個 FAQ）
8. 撰寫結尾 CTA 區塊（強化主要成果 + 重複 CTA）
9. 提供 SEO 元標題與描述建議

## 規則

### 必須
- 以客戶成果為導向，而非產品功能
- 全篇使用第二人稱（「您」）
- 標題不超過 10 個字；副標題不超過 20 個字
- 包含至少一個具體數據或佐證
- CTA 按鈕文字必須是「動作動詞 + 效益」的組合

### 禁止
- 使用目標受眾不熟悉的行話
- 在未說明效益的情況下撰寫功能導向文案
- 每個區塊包含超過一個主要 CTA
- 在標題中使用被動語態

## 範例

### 正確用法
```
[英雄區塊]
標題：不做繁瑣行政，專注成交更多客戶
副標題：專為創辦人打造的 CRM——自動追蹤每一次跟進，讓您聚焦在銷售上。
CTA：免費開始使用——無需信用卡
```

### 錯誤用法
```
[英雄區塊]
標題：我們的平台運用先進的 AI 技術
副標題：我們開發了一個解決方案，提供管理客戶關係的功能。
CTA：提交
```
