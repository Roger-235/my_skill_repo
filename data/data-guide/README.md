# Data Guide — 資料 Skill 導覽

引導你使用 data 分類下的 skill，說明範疇與觸發方式。

## 使用時機

詢問「有哪些資料 skill」、「資料庫 skill 怎麼用」、「database skill guide」時觸發。

## Skill 一覽

| Skill | 用途 |
|-------|------|
| `db-schema` | 資料庫 schema 設計：需求 → ER 圖 → DDL → Migration |

## db-schema 涵蓋範疇

✅ 涵蓋：ER 圖（Mermaid）、正規化（3NF）、DDL SQL、Migration（up + down）、Index 建議、PII 標記

❌ 不涵蓋：ORM 模型生成（Prisma / SQLAlchemy）、查詢最佳化、ETL / 資料管線
