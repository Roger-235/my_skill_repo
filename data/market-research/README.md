# Market Research — 市場調查分析

結構化的市場調查框架，涵蓋競品分析、SWOT 分析，以及 TAM/SAM/SOM 市場規模計算，產出可直接用於決策的研究報告。

## 功能

- 定義研究範疇（市場、地理區域、時間軸、決策目的）
- 識別並分類競爭對手（直接競品、間接競品、替代方案）
- 針對主要競品進行深度分析（強弱項、定價、目標客群、近期動態）
- 執行 SWOT 分析（優勢、劣勢、機會、威脅）
- 計算三層市場規模：TAM（全市場）、SAM（可服務市場）、SOM（可獲取市場）
- 識別 2–4 個主要客戶分群，含痛點與購買觸發條件
- 明確標示資料缺口與假設前提，不以猜測填補空白

## 觸發時機

**手動觸發**：
- 「市場調查」、「競品分析」、「分析市場」、「商業分析」
- market research、competitive analysis、SWOT analysis、TAM SAM SOM
- competitor research、market analysis

**不觸發**：
- 財務建模或收入預測 → 使用 `financial-modeling`
- 使用者研究或 UX 分析 → 使用 `ui-ux-pro-max`

## 使用方法

直接描述要調查的市場或產品，並說明此研究要協助做出的決策。例如：

> 「幫我做一份 AI 程式碼審查工具的市場調查，對象是大型企業工程團隊，用來評估是否自行開發。」

Claude 會依序完成：範疇定義 → 競品識別 → 競品分析 → SWOT → 市場規模 → 客戶分群 → 彙整報告。

## 前置需求

- 明確的研究主題（產品或商業領域）
- 此研究要協助做出的決策（若未提供，Claude 會先詢問）
- 公開市場資料的存取能力（網路搜尋、產業報告或使用者提供的資料）

## 輸出格式

報告包含以下章節：
- **Scope**：研究範疇與決策背景
- **Competitive Landscape**：競品比較表
- **SWOT Analysis**：四象限分析
- **Market Size**：TAM / SAM / SOM（含來源與假設）
- **Customer Segments**：客戶分群描述
- **Key Findings**：三大核心洞察
- **Data Gaps**：明確標示的資料缺口
