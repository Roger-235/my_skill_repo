# Design Guide — 設計 Skill 導覽

引導你使用 design 分類下的 skill，說明其涵蓋的能力與觸發方式。

## 使用時機

詢問「有哪些設計 skill」、「UI skill 導覽」、「設計 skill 怎麼用」時觸發。

## Skill 一覽

| Skill | 用途 |
|-------|------|
| `ui-ux-pro-max` | 50 種視覺風格 × 9 種技術棧，完整 UI/UX 實作 |

## ui-ux-pro-max 能力速查

| 需求 | 關鍵字 |
|------|--------|
| 選技術棧 | React / Next.js / Vue / Svelte / SwiftUI / Flutter / HTML+Tailwind / shadcn |
| 選視覺風格 | glassmorphism / brutalism / neumorphism / bento grid / dark mode / minimalism… |
| 選配色 | 21 種調色盤 |
| 選字型 | 50 種字型配對 |
| 圖表 | 20 種圖表類型 |
| 審查 / 修改現有 UI | "review", "fix", "improve", "optimize" |
