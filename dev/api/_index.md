# dev/api — API 與資料庫

| Skill | 說明 | 觸發詞 |
|-------|------|--------|
| api-design-reviewer | 審查 REST/GraphQL/RPC API 設計的一致性、安全性與開發者體驗，產出嚴重性分級報告 | review API design, audit API, check API spec, evaluate REST API, review OpenAPI spec |
| api-test-suite-builder | 為 API 端點生成涵蓋 Happy Path、驗證、認證、邊界條件與安全性的完整測試套件 | build API tests, generate API test suite, write API tests, test this API, create integration tests |
| database-designer | 設計完整資料庫架構：類型選型、資料模型、索引策略、分割、複寫與備份方案 | design database, database architecture, choose database, design data model, relational vs NoSQL |
| database-schema-designer | 產出生產就緒的資料庫 Schema DDL：資料表、約束、索引與 UP/DOWN 遷移腳本 | design schema, write DDL, create table, database schema, write migration, model this data |
