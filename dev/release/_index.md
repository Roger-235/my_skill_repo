# dev / release — 發布管理

| Skill | 說明 | 觸發詞 |
|-------|------|--------|
| changelog-generator | 從 git 歷史/Conventional Commits 產生 Keep a Changelog 格式的 CHANGELOG.md | changelog, release notes, what changed, conventional commits, keep a changelog, version history |
| release-manager | 完整軟體發布流程：SemVer 版本決策、標籤、CI 驗證、產出物發布、發布後驗證 | release, cut a release, ship, publish, semantic versioning, version bump, release branch |
| runbook-generator | 操作 Runbook 產生器：部署、事故應對、維護、擴縮容、還原程序 | runbook, operational guide, sop, standard operating procedure, on-call runbook, deployment runbook |
