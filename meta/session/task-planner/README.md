# Task Planner — 任務規劃器

將複雜目標分解為依賴有序、優先排列的原子化子任務，整合 TodoWrite 追蹤進度，確保複雜工作有條不紊地推進。

## 功能

- 釐清目標與可驗證的完成條件（若目標模糊，僅問一個關鍵問題）
- 將目標分解為原子化子任務，每個子任務可在單一工作階段完成
- 識別任務間的依賴關係並建立明確的執行順序
- 優先執行高不確定性、高風險任務以儘早驗證假設
- 將所有子任務寫入 TodoWrite，確保進度隨時可見
- 在執行過程中維護任務狀態，範疇變更時明確重新規劃

## 觸發時機

**手動觸發**：
- 「分解任務」、「制定計畫」、「任務規劃」、「幫我規劃」
- plan this task、break down task、create a plan、task decomposition
- how should I approach this

**自動觸發**：
- 任何具有 3 個以上不同步驟、尚未分解的任務

**不觸發**：
- 單一步驟的簡單任務
- 已有完整規劃與待辦清單的任務

## 使用方法

描述你想要完成的目標，Claude 會自動拆解規劃。例如：

> 「幫我規劃如何建立一個使用者認證系統，包含 JWT 和資料庫儲存。」

Claude 會先確認目標和完成條件，再分解子任務、建立依賴關係、寫入 TodoWrite，最後從第一個未被阻塞的任務開始執行。

## 前置需求

- 明確的目標陳述（若模糊，Claude 會先詢問一個關鍵問題）
- TodoWrite 工具可用於建立和追蹤子任務

## 輸出格式

輸出包含：
- **Goal & Done when**：目標重述與可驗證的完成條件
- **Subtask List**：包含依賴關係、優先級和完成條件的子任務表
- **Execution Order**：按依賴順序排列的執行序列
- **Starting now**：立即開始的第一個任務
- **TodoWrite**：所有子任務同步寫入追蹤系統
