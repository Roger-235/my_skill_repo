# Ops Skill Guide — 運維技能導覽

路由技能，幫助用戶找到正確的 ops 類 skill，不直接執行任何 CI/CD 設定或部署工作。

## 功能

- 根據用戶的運維目標推薦正確的 skill
- 說明各 skill 的觸發方式與適用場景
- 展示 CI/CD 設定到正式部署的完整 skill 鏈

## 觸發時機

- 「有哪些 ops skill」、「部署 skill 怎麼用」、「CI/CD 怎麼用」
- "which ops skill", "ops skill guide"

**不觸發**：實際設定 CI/CD 或執行部署時，直接路由至對應 skill。

## 使用方法

說明運維目標後，guide 會推薦：
- **ci-cd-pipeline** — 設計與設定 CI/CD pipeline（GitHub Actions、GitLab CI 等）
- **deploy-ops** — 正式環境部署清單、健康檢查與回滾指南

## 前置需求

無。
