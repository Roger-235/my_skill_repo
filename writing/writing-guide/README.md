# Writing Skill Guide — 寫作技能導覽

路由技能，幫助用戶找到正確的寫作類 skill，不直接執行任何寫作工作。

## 功能

- 根據用戶的寫作目標推薦正確的 skill
- 說明各 skill 的觸發方式與適用場景
- 展示 skill 組合鏈（如需要旗艦文章加社群內容）

## 觸發時機

- 「有哪些寫作 skill」、「writing skill 怎麼用」
- "which writing skill", "writing skill guide"

**不觸發**：實際撰寫文章或生產內容時，直接路由至對應 skill。

## 使用方法

說明寫作目標後，guide 會推薦：
- **article-writing** — 撰寫結構化技術文章或部落格文章
- **content-engine** — 從一個核心主題批量衍生多種格式內容

## 前置需求

無。
